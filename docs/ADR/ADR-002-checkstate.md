# ADR-002: 勾选状态存 localStorage 而非后端

## Status
Accepted

## Context
Check Sheet 需要记录「用户已掌握哪些术语」。选项：
- 存浏览器 `localStorage`（无后端）
- 存后端数据库（需账号体系）

## Decision
MVP 阶段将勾选状态存于浏览器 `localStorage`，并提供「导出 / 导入 JSON」做备份。

## Consequences
- **更易**：无需用户系统、无需服务端，前端开箱即用。
- **更难**：换设备 / 清缓存会丢失；多人无法共享同一份勾选。
- **缓解**：提供导出/导入；未来若需共享，可加可选同步层（不影响现有数据模型）。

## 复核触发条件
出现「用户要求跨设备同步」或「团队要统一看板」时，引入可选同步/后端。
